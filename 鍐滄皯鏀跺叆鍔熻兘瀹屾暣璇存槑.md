## 🔧 技术细节

### API调用
```javascript
// 获取用户列表（买家信息）
GET /api/auth/users
Authorization: Bearer <token>

// 返回格式
{
  "users": [
    {
      "id": "xxx",
      "username": "张三",
      "email": "zhangsan@qq.com",
      "role": "consumer"
    }
  ]
}
```

### 收入计算逻辑
```javascript
// 已到账 = paid + shipped + finished
const paidEarnings = myOrders
  .filter(o => ['paid', 'shipped', 'finished'].includes(o.status))
  .reduce((total, order) => {
    const myItems = order.items.filter(item =>
      myProducts.some(p => p.id === item.productId)
    );
    return total + myItems.reduce((sum, item) =>
      sum + item.price * item.qty, 0
    );
  }, 0);

// 待到账 = pending
const pendingEarnings = myOrders
  .filter(o => o.status === 'pending')
  .reduce(...); // 同上
```
